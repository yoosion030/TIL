# \<form>태그

## \<form>태그는 웹 페이지에서의 입력 양식을 의미한다.

### form 관련 속성들

> name : 폼의 이름 지정 <br>
> action : form이 전송되는 백엔드 url <br>
> method : 폼 전송 방식 <br>
> autocomplete : 자동완성기능<br>
> target : action에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에 열도록 지정<br>

### form 관련 태그들

> input <br>
> select <br>
> option <br>
> label <br>
> button <br>
