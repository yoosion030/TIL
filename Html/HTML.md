# HTML 문서의 구조
##  HTML ( HyperText Markup Language ) 정의
> 웹 문서를 만드는 기본언어

## \<!DOCTYPE html> 태그
> 웹 브라우저에 현재 문서가 HTML5문서라고 알려주는 것

## \<html> 태그 
- \<html> : 웹 문서가 시작된다는 뜻
- \</html> : 웹 문서가 끝났다는 뜻
- \</html> 뒤에는 아무 내용도 없어야 함

## \<head> 태그
- 웹 브라우저가 알아야 할 정보를 입력하는 곳
- head 영역의 내용은 대부분 웹 브라우저 화면에는 보이지 않음
- \<meta> : 문자 세트를 비롯해 문서 정보가 들어있음
- \<title> : 문서의 제목을 나타냄

## \<bddy> 태그
- 실제 웹 브라우저에 표시할 내용을 입력
