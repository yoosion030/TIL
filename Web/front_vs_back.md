# 프론트 엔드 ( FRONT-END )
- 사용자의 앞에 보이는 영역
- 웹 브라우저 화면에 보이는 것을 다룸
- 웹 페이지를 그리는 기술

## 프론트 엔드 개발 언어
- HTML
- CSS
- Javascript
## 프론트 엔드 기술
- ReactJS
- Bootstrap
- AngularJS
# 백 엔드 ( BACK-END )
- 사용자의 뒤에서 보이지 않는 영역
-  UI나 GUI로 구성된 화면의 통신이나 요청에 대하여 DB나 인터페이스 등을 통해 시스템 구성 실체에 접근하는 것
-  데이터를 저장하고 관리함
-  클라이언트 측에서 모든 것이 매끄럽게 작동할 수 있게 해줌
## 백 엔드 개발 언어
- Java
- Node.js
- PHP
- Python
## 백 엔드 기술
- Express
- Django
