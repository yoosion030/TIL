# NextJS란?
**NextJS**는 따로 설정을 해주지 않고도 SSR, SEO부터 TypeScript까지 생산에 필요한 많은 기능들을 제공하는 아주 강력한 React 프레임워크 이다.

## NextJS 특징

1. 직관적인 라우팅 시스템 ( dynamic routes 지원, ex: /posts/:id )
2. 페이지를 자동으로 최적화
3. 데이터가 필요한 페이지를 SSR 할 수 있게 지원
4. 빠른 페이지 로드를 위한 자동 코드 스플라이팅
5. HMR을 지원
6. 쉬운 커스텀마이징을 할 수 있음


## NextJS 핵심기능
1. SSR
2. 파일기반 라우팅 등등 .. 엄청 많음
